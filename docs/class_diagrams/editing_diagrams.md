## Using PlantUML in PyCharm

PlantUML allows you to create UML diagrams from plain text. Here’s how to get started in PyCharm:

1. **Install the Plugin**
   - Go to `Settings` → `Plugins`.
   - Search for **PlantUML Integration (`plantuml4idea`)** and install it.
   - Restart PyCharm if prompted.

2. **Create a PlantUML File**
   - Add a new file with the `.puml` extension.
   - Write your UML code

3. **Preview Your Diagram**
   - Open the `.puml` file.
   - The plugin automatically renders the diagram in a side panel.
   - If the preview doesn’t appear, right-click the editor and select `Show PlantUML Preview`.

4. **Exporting**
   - You can export the diagram as PNG, SVG, or PDF via the preview panel toolbar.

5. **Tips**
   - Keep diagrams small and modular in separate `.puml` files for easier maintenance.
   - Refer to [PlantUML documentation](https://plantuml.com/) for advanced features like sequences, class diagrams, and notes.
